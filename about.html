Project Overview

This project is a web application designed to display football schedules, countdown, and betting data. It organizes teams by league, conference, and division, and dynamically loads team schedules and betting analysis.

Current Features

  • Dynamic list of NFL teams organized by conference and division
  • Display of team schedules with opponent and game times
  • Betting analysis with indicators for favorites and underdogs
  • Support for both NFL and college football data

Work in Progress

This project is a work in progress. Future updates will include additional features such as college football, weather data integration, affiliate links, and more detailed betting analysis.
